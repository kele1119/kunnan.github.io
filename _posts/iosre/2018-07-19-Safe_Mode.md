# 越狱机器的问题处理

#### 进入安全模式的方式

> * tweak 崩溃主动进入安全模式
>
> * 手动进入[安全模式](https://www.baidu.com/s?wd=%E5%AE%89%E5%85%A8%E6%A8%A1%E5%BC%8F&tn=SE_PcZhidaonwhc_ngpagmjz&rsv_dl=gh_pc_zhidao)：
>
>   ```
>   在关机状态下按睡眠键开机，屏幕亮以后松开睡眠键，按住任意音量键，即可进入安全模式
>   ```
>
>   

#### 技巧性解决tweak导致的问题

> * 使用越狱工具（yalu），重新越狱的时候进行tweak和文件的删除
